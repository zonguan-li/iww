# 设备同步工具

跨设备文件和消息同步工具，基于 GitHub 存储。

## 使用方法

1. 访问：https://zonguan-li.github.io/iww/file-sync.html

2. 首次使用会弹出配置窗口，需要填写：
   - **GitHub用户名**：你的 GitHub 用户名
   - **仓库名称**：你的仓库名称
   - **GitHub Token**：你的 GitHub Personal Access Token

   如需获取配置信息，请联系管理员。

3. 点击"保存"即可开始使用

4. 在 A 设备和 B 设备上都按照上述步骤配置即可实现同步

## 功能说明

- **消息发送**：在消息框输入文本，点击发送
- **文件上传**：拖拽文件到上传区域或点击上传
- **自动刷新**：每 1 秒自动同步一次
- **手动刷新**：点击 🔄 刷新按钮
- **清理数据**：点击 🗑️ 清理按钮删除所有消息/文件
- **单条删除**：每条消息/文件右侧有删除按钮

## 提示

- 配置信息保存在浏览器 localStorage 中，不会泄露
- 点击右上角"⚙️ 配置"可以重新配置
- 点击配置弹窗背景或按 ESC 键可关闭配置窗口
